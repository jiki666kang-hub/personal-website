# 康康求职行动台

这是康康99平台线上发布版里的求职行动台，内容来自现有简历、求职规划、每日计划和作品截图。

## 怎么打开

本地预览时直接打开：

```text
康康99平台线上发布版\index.html
```

## 它适合做什么

- 睡醒后确认今天最该做的 3 件事。
- 选择主攻求职方向。
- 检查面试前材料包是否齐全。
- 快速复制中文/英文自我介绍。
- 记录本机投递进展。
- 准备留服认证材料。

## 明天最小动作

只做一件事也可以：打开本工具，在“今日”页勾完第一项，然后在“投递”页新增 1 个目标岗位。

## 本次新增优化

- 左上角头像可以点击，进入“个人中心”。
- 新增“日历”页，可切换 2026 年 1-12 月，点击重点日期可以查看当天事项。
- “硬提醒”里的 2026-06-18 可以点击，会直接打开取证日程详情。
- “硬提醒”里新增 17/18/19/20 迷你日历按钮，可直接跳到对应日程。
- 日历详情里新增“个人备注”，每个日期都能保存自己的安排。
- “认证”页的勾选会自动保存到本机，并显示保存数量。
- “认证”页新增“导出进度”，可以把今日、材料、认证和投递进度导出为 JSON 备份。
- “认证”页新增“导入”，可以恢复之前导出的 JSON 备份。
- 新增“保存中心”，集中显示今日任务、材料包、认证材料、日历备注、投递记录的本机保存数量。
- 保存中心支持“保存自检”“导出备份”“导入 JSON 备份”“复制保存摘要”。
- “方向”页卡片可以点击，打开推荐简历、材料包、关键词和招聘网站入口。
- “方向”详情新增“复制搜索关键词”和“岗位匹配打分器”入口。
- “方向”详情新增按关键词快速搜索：BOSS、猎聘、智联、国聘。
- “个人中心”的材料入口按“主推简历、面试与认证、工具与来源”分组。

## 招聘链接说明

方向页里的招聘链接来自公开招聘平台或公司招聘页，用于快速打开搜索入口和参考岗位描述。招聘信息会变化，正式投递当天仍要以招聘网站显示的岗位状态为准。

已加入的入口包括：

- BOSS直聘 GNSS / 工程测量 / 俄语相关搜索页。
- 猎聘测绘工程师、测绘/测量、千寻位置招聘页。
- 智联招聘相关岗位搜索页。
- 国聘官网和俄语商务岗示例页。
- 天硕导航、和芯星通等 GNSS 公司招聘页。
