# 岗位筛选规则

## 看到这些词，优先点开

- GNSS
- RTK
- 测绘设备
- 国际技术支持
- 海外项目
- 俄语
- 俄罗斯 / 中亚 / 独联体
- 工程测量
- 测量员
- 地理信息

## 这些岗位适配度高

### GNSS / 国际技术支持

适配理由：

- 测绘本科和大地测量硕士背景匹配。
- 俄语 B2 和俄罗斯学习生活经历是差异化优势。
- 能接受长期出差和海外驻场。
- 项目经历能覆盖外业采集、内业处理、成果表达。

投递时强调：

- “测绘技术 + 跨语言沟通 + 海外适应”
- 不要只说会俄语，也不要只说会测绘。

### 工程测量 / 测绘技术员

适配理由：

- 天津地铁施工测量实践。
- 1:500 地形图完整流程。
- RTK、全站仪、CAD、CASS、数据整理。

投递时强调：

- “能吃现场节奏，重视复核和成果可追溯。”

### 俄语销售 / 海外客户服务

适配理由：

- 30 余名中国留学生落地服务。
- 熟悉俄罗斯学校、银行、运营商和行政流程。
- 展会翻译和跨文化沟通经历。

投递时强调：

- “客户接待、流程解释、问题跟进、服务转化意识。”

## 这些岗位今天先别花太多时间

### 纯算法轨道仿真

不是不能投，但今天不要作为主线。除非岗位明确接受应届生、助理研究、数据处理或仿真辅助。

### 纯软件开发

如果要求熟练 C++、工程化开发、算法部署，今天先跳过。

### 高级测绘项目经理

如果要求多年现场管理经验，今天先跳过。

## 判断一个岗位是否值得投

满足 3 条就可以投：

- 岗位提到测绘、GNSS、海外、俄语、工程测量中的至少 2 个。
- 不强制多年工作经验。
- 你能用现有简历和作品集解释自己为什么匹配。

如果只满足 1 条，先收藏，不要马上投。
