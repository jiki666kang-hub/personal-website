# 明天给 Codex 的提示词

如果你睡醒后想继续推进，直接复制下面这段给 Codex：

```text
请先读取康康99平台线上发布版里的求职行动台和求职行动包。
我今天的目标不是做很多事，而是完成一次真实求职推进。
请帮我按下面顺序做：
1. 从我的资料里选出今天最适合投的 3 类岗位；
2. 帮我筛选 5 个适合搜索的岗位关键词；
3. 帮我把一个岗位的投递材料包列出来；
4. 帮我准备一段给 HR 的中文开场消息；
5. 最后把今天只需要做的 3 件事写进每日计划。
回答要简单，一步一步来，不要一次讲太多理论。
```

如果你已经找到一个具体岗位，就复制这段：

```text
我找到一个岗位，岗位信息如下：
【把岗位要求粘贴在这里】

请根据我的简历和求职行动包判断：
1. 这个岗位匹配度高不高；
2. 应该用哪份简历；
3. 需要补哪 2 页作品集；
4. 给 HR 的第一条消息怎么写；
5. 面试最可能被问哪 5 个问题。
```

如果你感觉又乱了，就复制这段：

```text
我现在有点乱。请你先不要讲理论，只根据我的求职行动包，帮我选出接下来 30 分钟唯一要做的一件事，并告诉我第一步点开什么文件。
```
