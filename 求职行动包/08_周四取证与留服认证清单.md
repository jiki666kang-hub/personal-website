# 周四取证与留服认证清单

日期：2026-06-18 周四

## 去系办前

- [ ] 带护照。
- [ ] 带学生证或学校相关身份证明。
- [ ] 手机电量充足。
- [ ] 确认系办办公时间。
- [ ] 准备一句俄语说明：来取毕业证和硕士成绩单。

俄语表达：

```text
Здравствуйте. Я пришёл получить диплом и официальную академическую справку / транскрипт.
```

如果对方说还没好：

```text
Подскажите, пожалуйста, когда можно будет получить документы?
```

## 到系办后要确认

- [ ] 毕业证或学位证是否已经可领取。
- [ ] 硕士成绩单是否已经可领取。
- [ ] 姓名拼写是否与护照一致。
- [ ] 出生日期是否正确。
- [ ] 专业名称是否正确。
- [ ] 学习起止时间是否正确。
- [ ] 是否有学校盖章或签字。
- [ ] 是否需要另开英文/俄文版本。

## 拿到后马上做

- [ ] 用手机拍清晰照片，先留备份。
- [ ] 回住处后扫描成 PDF。
- [ ] 文件名按下面格式命名：

```text
汲康康_毕业证_原件扫描.pdf
汲康康_硕士成绩单_原件扫描.pdf
汲康康_护照信息页.jpg
汲康康_俄罗斯签证页.jpg
```

## 留服认证材料准备

- [ ] 学历学位证书彩色扫描件。
- [ ] 完整官方成绩单。
- [ ] 护照个人信息页。
- [ ] 俄罗斯学习签证页、电子签证或居留卡。
- [ ] 出入境记录，按系统提示通过国家政务服务平台 App 完成实人认证并获取。
- [ ] 近期证件照。
- [ ] 文凭证书核查授权声明。
- [ ] 如有补充要求，准备学习经历说明和学校联系方式。

## 提交前最后检查

- [ ] 不修改、不美化、不裁改证书或护照内容。
- [ ] 姓名拼写与护照一致。
- [ ] 证书编号、专业名称、日期与原件一致。
- [ ] 扫描件清晰、边缘完整。
- [ ] 每个文件大小符合系统要求。
- [ ] 保存申请编号、订单号和支付记录。

## 官网流程核对

教育部留学服务中心当前公开流程包括：

1. 注册账户并完成实名认证。
2. 登录服务大厅，办理认证申请。
3. 签署服务协议。
4. 用国家政务服务平台 App 完成实人认证，获取出入境记录。
5. 填写基本信息、申请信息和学习经历。
6. 上传申请材料。
7. 在线支付费用。
8. 在用户中心查看进度；如收到补材料通知，及时补交。

## 风险提醒

留服认证政策和系统页面可能调整。正式提交当天，以教育部留学服务中心网上服务大厅显示的材料框和流程为准。
