## 上传到 GitHub 后

在仓库的 `Settings -> Pages` 中开启 GitHub Pages：

- Source 选 `Deploy from a branch`
- Branch 选 `main`
- Folder 选 `/root`

保存后等待一两分钟，GitHub 会生成一个可访问链接。
